## Install

```sh
$ npm install @chialab/synapse
# or
$ yarn add @chialab/synapse
```

## Development
### Requirements

* Node (>= 8.9)
* RNA cli (`npm install @chialab/rna-cli`)


### Build the project

```
$ rna install
$ rna build --production
```

### Watch the project
```
$ rna install
$ rna build --watch
```
